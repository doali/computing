# MAC address

- Media Access Control Address
- physical address
- hardware address
- and sometimes ethernet address
- burnt into the device
- 48 bits
- FF:FF:FF:FF:FF:FF broadcast
- `arp -a` list all ip addresses and their physical addresses
- `rarp ...`
