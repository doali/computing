# Certificats et chiffrements

## Vocabulaire

Les termes suivants sont équivalents 
- hash,
- checksum (MD2, MD4, MD5, RIPEMD-160)
- message digest (SHA-1 Secure Hash Algorithm, SHS Secure Hash Standard)
- condensât
